# **App Name**: ScanToDB

## Core Features:

- Barcode Scanning: Use the phone's camera to scan barcodes.
- Data Input: Allow manual input of data fields (in addition to barcode scanning).
- Supabase Integration: Connect to a Supabase database to store the scanned data.
- Real-time Data Sync: Ensure that data is synced in real-time to the Supabase database.
- Data Validation: Apply validation rules to the scanned data before storing it in Supabase.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for a professional and reliable feel.
- Background color: Light gray (#F0F4F8) to provide a clean and modern backdrop.
- Accent color: Soft green (#81C784) to highlight key actions and success states.
- Body and headline font: 'Inter' sans-serif for clear readability and a modern aesthetic.
- Use simple, consistent icons for barcode scanning and data entry actions.
- Design a clear, intuitive layout that prioritizes barcode scanning and manual data entry.
